Self hosting

Would you like to host mySE yourself? Feel free to download the source code and setup your own server.

There are only 3 steps to setup your server.

  1. Make sure you have python 3.x.x installed
  2. Run the setupDB.py file.
  3. Run the server.py file.

Application Integration

Would you like to create your own custom mySE application? That is possible, just install the server and create your custom app. You are free to use any License for your program, the original files (or modified) will still be on a MIT license. You can find our API documentation here. WARNING: The API docs are NOT mobile friendly.
