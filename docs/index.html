mySE

My Secure Environment

What is mySE?

mySE stands for 'My Secure Environment'! mySe was created as a project for GO-AO.

What does mySE do?

mySE reports who passed the room and if the light was on or off.

How does mySE work?

When this event occurs the Arduino sends a signal to a server which can then save the log. When the user requests the logs the server fetches these from the database.
