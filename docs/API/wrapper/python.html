Install guide:

To install the LTS mySE module you can use PIP!

            >>> pip install mySE

A basic example:

Example code, this code creates and/or fetches the token from the API.
Then it creates a Client Object and retrieves the id. 

            
                from mySE import mySE

                local_url, mail, token = "http://127.0.0.1:5000/", "mail@mail.mail", None

                try:
                    token = mySE.gen_token(local_url, mail)
                except mySE.error.UnauthorizedError:
                    token = mySE.get_token(local_url, mail)

                client = mySE.Client(local_url, token)
                print(f"Client ID: {client.id}")

API reference:

mySE.mySE.Client(url, token)

Represents a mySE Client, this is the main object for this API wrapper.
This is object is used to interact with the mySE API.

Parameters:

  • url (Required) -
    The mySE server link. Eg. use "http://127.0.0.1:5000/" if you're running the server locally on port 5000.
  • token (Required) -
    Your unique mySE token. If you don't have a token yet generate it using gen_token.
